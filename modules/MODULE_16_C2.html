Section 1: What a C2 Framework Is

A C2 (Command and Control) framework is the operator's interface to a compromised machine. The implant (payload on target) calls back to the C2 server. The operator sends commands, receives output. Everything encrypted, jittered, over normal-looking traffic.

The implant never opens a visible window. The C2 port (4443) looks like HTTPS to network monitors. Jitter randomises beacon timing so it doesn't look like a heartbeat.
