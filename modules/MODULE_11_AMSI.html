Section 1: What AMSI Is

AMSI (Antimalware Scan Interface) is a Windows API that lets AV engines scan content before it executes. Every PowerShell script, VBA macro, and .NET assembly gets passed to AMSI before running. Defender hooks into it.

PowerShell script → AmsiScanBuffer() → Defender engine → BLOCK / ALLOW VBA macro → AmsiScanBuffer() → Defender engine → BLOCK / ALLOW .NET assembly → AmsiScanBuffer() → Defender engine → BLOCK / ALLOW AmsiScanBuffer lives in: C:\Windows\System32\amsi.dll It's loaded into every process that calls it — including your implant's process.

Section 2: HWBP — The Zero-Write Alternative

Hardware Breakpoints (HWBP) use CPU debug registers — DR0–DR3 for addresses, DR7 for control. When the CPU hits an address in DR0, it fires a debug exception before executing that instruction. Your VEH (Vectored Exception Handler) catches it first.

Normal path (no bypass): PowerShell → AmsiScanBuffer(0x...8160) → scans string → EDR blocks HWBP bypass path: 1. Set DR0 = address of AmsiScanBuffer 2. Set DR7 = 0x405 (enable DR0 execute breakpoint) 3. Register VEH handler 4. PowerShell → CPU hits DR0 → fires EXCEPTION_SINGLE_STEP 5. VEH handler fires BEFORE AmsiScanBuffer executes 6. Handler modifies CONTEXT.Rax = AMSI_RESULT_CLEAN (1) 7. Handler skips past AmsiScanBuffer → returns 1 to caller 8. Defender engine sees: CLEAN → allows execution Memory writes: ZERO. Nothing patched. EDR memory integrity check: PASS.

Section 3: ETW — Blind the EDR Too

ETW (Event Tracing for Windows) is how EDRs see what's happening. Every process reports events via EtwEventWrite. If you blind ETW, the EDR goes dark — it can't see your process's activity even if AMSI is bypassed.

# Same HWBP technique — second register
HMODULE hNtdll = GetModuleHandleA("ntdll.dll");
FARPROC pEtw = GetProcAddress(hNtdll, "EtwEventWrite");

ctx.Dr1 = (DWORD64)pEtw;    // DR1 = EtwEventWrite
ctx.Dr7 = 0x405 | 0x400;   // enable DR0 + DR1

// VEH catches both:
// DR0 hit → fake AMSI result
// DR1 hit → skip EtwEventWrite → event never reaches EDR
