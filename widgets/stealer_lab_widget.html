A safe, browser-based simulation of how credential-stealing malware phones home to a C2 server. No real data is accessed.

EDUCATIONAL USE ONLY. This widget simulates the network protocol used by real infostealers. The full lab code is available in the private 22div-stealer-lab repository for authorized 22DIV students.

  • Stealer reads Login Data / Cookies
  • Stealer checks in with C2 (POST /create_log)
  • Stealer sends credentials (POST /log_data)
  • Stealer uploads files (POST /log_files)
